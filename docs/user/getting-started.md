# Getting Started

## What It Does

LocalVoice lets you dictate text into any desktop application entirely offline, using local transcription models and sidecar runtimes.

## First Launch

When you start LocalVoice for the first time:

1. The main window opens by default, including when setup is needed.
2. The default recording overlay appears only while you record.
3. You can choose **Default view -> Tray only** if you prefer tray-only startup.

## Recording Overlay

Press the global shortcut to start recording. The default pill mode shows a small bottom-center waveform overlay while recording and hides it after you stop.

The overlay does not show transcript text. Final text still goes through cleanup, dictionary rules, history, clipboard, and optional auto-insert.

## Classic Pill

If you prefer the older persistent pill:

1. Open **Settings -> Appearance**.
2. Set **Pill mode** to **Classic pill**.
3. Optionally set **Default view** to **Classic pill**.

Classic pill can be dragged, expanded, and used for quick actions such as Copy, History, and Settings.

## The Full Window

Open the full window from the tray icon or from Classic pill. The full window gives you access to Dashboard, History, Dictionary, Models, and Settings.

## Tray Icon

Right-click the tray icon for quick actions such as recording, Dashboard, History, Settings, and Quit. In recording overlay mode, left-click opens the main window. In Classic pill mode, left-click toggles the pill.
