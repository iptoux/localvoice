# Dashboard

## What It Does

The Dashboard shows your dictation usage at a glance — how many words you've transcribed, how many sessions you've recorded, your average speaking speed, and how your usage has changed over time.

## Metrics

| Card | Meaning |
|------|---------|
| **Total Words** | Sum of word counts across all sessions in the selected date range |
| **Sessions** | Number of completed transcription sessions |
| **Avg WPM** | Average words-per-minute across sessions that have a duration |
| **Recording Time** | Total duration of all sessions combined (h:mm format) |

## Charts

### Words over time

A line chart showing how many words you transcribed each day (or week, for All time). Useful for spotting usage patterns.

### Language breakdown

A horizontal bar chart showing how many sessions were recorded in each language. The colour per language is consistent across the app.

## Date Range

Use the **Last 7 days / Last 30 days / All time** selector in the top-right corner to change the period for all metrics and charts at once.
