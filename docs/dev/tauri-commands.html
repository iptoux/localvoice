Tauri Command API

Complete reference for all invoke() targets. Every command returns CmdResult<T> (Result<T, String>). Events are emitted via tauri::emit() and subscribed with listen().

Recording

start_recording

Starts a new audio recording session.

invoke('start_recording'): Promise<void>
async
Reads recording.device_id, recording.silence_threshold, recording.silence_timeout_ms from settings. Begins microphone capture. Transitions pill to Listening. Transcription runs in the background.

stop_recording

Stops the active recording, saves a WAV file, and triggers background transcription.

invoke('stop_recording'): Promise<string> // WAV file path
async

cancel_recording

Discards the current recording buffer and returns to Idle without saving.

invoke('cancel_recording'): Promise<void>

get_recording_state

Returns the current recording state.

invoke('get_recording_state'): Promise<RecordingState>
RecordingStateenum"idle" | "listening" | "processing" | "success" | "error"
